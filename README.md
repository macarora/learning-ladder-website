# Learning Ladder 🎮📚

A full educational gaming website built as part of a Web 
Technologies course at University of Newcastle.

## Overview
Learning Ladder is an educational platform designed to make 
learning fun through games. Features STEM and Literary game 
categories with a responsive design.

## Features
- Home page with featured educational games
- STEM and Literary game category pages
- About Us, Contact Us, Privacy Policy and Terms pages
- Cookie-based last visit tracking
- Responsive design for mobile and desktop
- Custom PHP theme built on GetSimple CMS

## Tech Stack
PHP · CSS · HTML · XML · GetSimple CMS

## Course
SENG1050 - University of Newcastle, 2023
